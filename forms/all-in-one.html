すべてのフォーム一覧

このサイトはHTMLフォーム属性の学習用サンプルです。実際のメールアドレス、電話番号、クレジットカード番号、パスワード、OTPを入力しないでください。入力内容は送信・保存されません。

中心に置くのは autocomplete 属性です。自動補完はブラウザー、OS、保存済みプロファイル、パスワードマネージャー、決済情報、ユーザー設定に依存します。

主要な推奨HTML

<input type="email" autocomplete="email" name="email" id="email">
<input type="tel" autocomplete="tel" name="tel" id="tel">
<input type="text" autocomplete="username" name="username" id="username">
<input type="password" autocomplete="current-password" name="current-password" id="current-password">
<input type="password" autocomplete="new-password" name="new-password" id="new-password">
<input type="text" inputmode="numeric" autocomplete="one-time-code" name="otp" id="otp">
<input type="text" inputmode="numeric" autocomplete="cc-number" name="cc-number" id="cc-number">
<input type="text" autocomplete="off" name="serial-number" id="serial-number">
